Ejercicio 1
===========

Desarrollar un programa para
calcular el impuesto y total
de una venta.

Los datos de entrada son:

- Precio del producto.
- La cantidad vendida.


Ejercicio 2
===========

Desarrollar un programa para
encontrar el promedio de un
estudiante de Java, las notas
son:

  - promedio de practicas (40%)
  - examen parcial (20%)
  - examen final (40%)

Aprueba si el promedio es mayor o
igual a 14.0.



Ejercicio 3

Ejercicio 4
============

Modificar el Ejercicio 3 para que se
procesen 5 números.


Ejercicio 5
===========

Desarrollar un programa para encontrar el 
promedio de un estudiante de Java, las notas
son:

  - 4 practicas calificadas,
    se promedian las 3 mejores. (40%)
  - examen parcial (20%)
  - examen final (40%)
  
El promedio final:
  pp * 0.40 + ep * 0.20 + ef * 0.40

Aprueba si el promedio es mayor o igual a 14.0.

Se deben mostrar todas las notas y la 
condición de aprobado o desaprobado.
